Guide to deploy new posts to pages from Hugo

Posted on Nov 1, 2023

To create a new blog post navigate to the blog directory and just

hugo new posts/<postname>.md

Then edit the generated markdown file with what ever you want.

Then save that new post and head and run

hugo -t archie

to generate the public files.

Then cd into the public directory and add and commit those new changes to the MAIN branch.
